# 🧪 Automated E2E Testing Suite

A robust end-to-end testing framework built with Cypress to automatically verify the core functionalities of a web application. This project ensures high software quality by catching regressions in both the UI and underlying APIs.

## 🚀 Key Features
* **Full Flow Coverage:** Tests critical user journeys from login to complex data manipulation.
* **BDD Approach:** Scenarios are written using Behavior-Driven Development principles for clear, readable test cases.
* **UI & API Integration:** Combines frontend element validation with backend REST API interception and mocking.

## 🛠️ Tech Stack
* **Framework:** Cypress
* **Language:** JavaScript
* **Methodology:** BDD
